ProtPainter 设计工作台

这个页面现在承担整个项目的第一阶段入口。当前已经接通的是 `Curve` 生成、筛选和可视化；后续 `Sketch / Backbone / Sequence / Folded Structure / Evaluation` 先以工作台形式展示，方便你把整条设计链路放进同一个网页里。

项目阶段

Stage 1
Curve
AE + OT 生成曲线，做几何筛选，并从保留结果里挑选候选。
Stage 2
Sketch
把 curve 转成 sketch token / 结构草图，作为后续结构生成条件。
Stage 3
Backbone
根据 sketch 生成 backbone / 结构候选，得到标准 PDB 级结果。
Stage 4
Sequence
基于 backbone 做序列设计，得到序列候选和设计分数。
Stage 5
Folded
用 OmegaFold 等模型把设计序列折回三维结构，并为后续一致性指标提供输入。
Stage 6
Evaluation
Sequence / Folded 完成后自动比较 curve / sketch / folded structure，做 RMSD、TM、pLDDT 等评估与选优。

Curve 参数

当前固定使用默认 AE checkpoint。`ot_bat_size_n` 主要影响 OT 分批和显存占用，通常应小于等于 `num_gen_x`。这块对应整个项目的 `Curve Stage`。
